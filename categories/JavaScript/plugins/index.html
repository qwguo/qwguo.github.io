cropper.js 图片处理插件

cropper.js一个用来处理图片的插件，可以使用它来实现图片的各种模式下的裁切效果，当我们在做一个上传头像或者上传图片功能的时候，需要用户裁切出用户想要的图片位置就可以利用这个插件来实现

cropper.js支持移动设备的图片剪裁。它基于HTML5 canvas，可以通过Base64编码导出剪裁后的图片。
